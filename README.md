# InvestNavigator AI / ИнвестНавигатор ИИ

ИнвестНавигатор ИИ — web-платформа инвестиционного анализа с элементами искусственного интеллекта.

## Цель проекта

Создать приложение, которое собирает финансовые данные, рассчитывает аналитические показатели, оценивает риски и формирует объяснимый AI-отчёт по выбранному активу.

## Основные модули

- Backend: Java + Spring Boot
- Frontend: React + TypeScript + Vite
- Database: PostgreSQL
- AI Service: Python FastAPI, later
- Deployment: Docker Compose

## MVP

В первой версии приложение должно поддерживать:

- список активов;
- карточку актива;
- демо-котировки;
- базовые метрики;
- AI-отчёт;
- сравнение активов.
